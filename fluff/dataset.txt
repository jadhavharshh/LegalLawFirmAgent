| Dataset                                                      | What it Contains                                                                                     | Why It’s Useful                                                         |
| ------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------- |
| **Indian Supreme Court Judgments (1950-2024)**               | Huge set of SC judgments in English + regional, with metadata. ([Open Data Registry][1])             | Good source of full case texts for RAG or fine-tuning.                  |
| **MILDSum**                                                  | \~3,122 Indian court judgments + summaries in English + Hindi. ([GitHub][2])                         | Useful for summarization tasks and multilingual summarization.          |
| **IndicLegalQA**                                             | \~10,000 Q\&A pairs from \~1,256 Supreme Court judgments, civil + criminal mix. ([Mendeley Data][3]) | Very useful for supervised fine-tuning with instruction + answer pairs. |
| **Indian Court Decision Annotated Corpus** (by Harde et al.) | \~50 SC decisions annotated for NER etc. Various encoding formats. ([遇见数据集][4])                      | Good if you want entity extraction / metadata tasks.                    |
| **Open Justice India / High Court Judgments**                | Judgments + orders across High Courts, some regional languages, large volume. ([Open Justice][5])    | Great for breadth, various jurisdictions.                               |
